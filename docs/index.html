Robot Auto-Balanceado: PID vs PID potenciado por IA

Material educativo para ejecutar una simulacion 2D de un robot auto-balanceado, comparar un PID clasico contra un PID con mezcla de expertos, revisar graficas, animacion y fundamentos teoricos.

Guia para estudiantesRequisitos, comandos de ejecucion, errores comunes y actividad de entrega. Compendio interactivoGuia, metricas, graficas y animacion en un solo HTML. PDF teoricoModelo de pendulo invertido, PID, PID IA y conexion con Gazebo. AnimacionVisualizacion lado a lado del PID normal y PID potenciado por IA.

Codigo fuente y comandos en el repositorio. Para ejecutar localmente: python scripts/sim_2d_pid_ai.py.
